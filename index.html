Speedy Reader

Данная страница поможет вам читать тексты бысрее, чем вы делаете это обычно.

Несколько советов перед ипользованием:

  1. Используйте Google Chrome и хромоподобные браузеры
  2. После каждого использования перезагружайте страницу
  3. Для полноты эффекта перейдите в полноэкранный режим:

    3.1 Windows: Нажмите клавишу F11

    3.2 Mac OS: Используйте сочетание клавиш shift + ⌘ command + F

    P.s. Для выхода из полноэкранного режима используйте те же клавиши

  4. Не вставляйте слишком большой текст
